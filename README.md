# ArizaBildirimTalepSureci
Arıza Bildirim Talep Süreci
Süreci her kullanıcı başlatabilir
1. AMAÇ
PaperWork uygulamasında "Arıza Bakım Talep Süreci"ni standartlaştırarak, arıza taleplerinin hızlı, doğru ve takip edilebilir şekilde yönetilmesini sağlamak. Hat duruşları, güvenlik sorunları ve diğer acil/günlük arızaların minimum sürede çözülmesini hedefler.
2. KAPSAM
Şirket bünyesindeki tüm departmanlar, atölyeler ve ekipmanlar için geçerlidir. ERP sistemiyle entegre PaperWork platformu üzerinden yürütülür.
